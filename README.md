## Overview
**Tiny Treasures** is a fully functional e-commerce website focused on 3D miniature frames.  
Our platform allows users to browse and customize products online according to their preferences, make secure purchases, and enjoy home delivery.  
This project was developed as part of our 2nd-year software project for our clients [Tiny Treasure].

The website is designed to be user-friendly, visually attractive, and fully responsive, ensuring a smooth shopping experience for customers.

---

## Features

### Customer Features
- **Product Browsing:** Explore all 3D miniature products.
- **Product Customization:** Customize products online with personal preferences.
- **Add Reviews & Ratings:** Submit reviews and view all reviews for products.
- **Wishlist:** Add products to wishlist for future reference.
- **Shopping Cart & Checkout:** Add products to cart, apply coupon codes, and make payments securely.
- **Home Delivery:** Get products delivered to your preferred address.
- **Search & Filters:** Search for products and filter based on preferences.
- **Profile Management:** Maintain account details, save multiple addresses, view past orders, and review history.
- **Dynamic Pricing:** Prices update dynamically based on selected customization options.
- **Notifications & Subscription:** Subscribe to the website to receive updates on new products, offers, and discounts.
- **Social Login:** Sign up and log in using Google, Facebook, and Instagram accounts.
- **Product Galleries:** Multiple images and zoom previews for better product visualization.
- **Coupons & Discounts:** Apply coupon codes to receive discounts.
- **Terms & Conditions, About Us, Return & Refund Policies:** Fully integrated informational pages.

### Admin Features
- **Admin Panel:** Manage products, advertisements, orders, and user reviews.
- **Role-Based Authentication:** Secure access to admin and customer features.
- **Order & Review Management:** Monitor customer purchases and submitted reviews.

## Technologies Used
- **Frontend:** Next.js, React, Tailwind CSS, TypeScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT with Google, Facebook, Instagram OAuth
- **Payment Gateway:** Stripe / Online Card Payments
- **APIs & Integrations:** Email notifications, dynamic product customization
- **Version Control:** Git & GitHub
